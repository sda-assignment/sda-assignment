# sda-assignment

## Prerequisites
Java 19 (jdk-19)

## Building and running
### On a UNIX/Linux environment in the root directory
```sh
$ ./mvnw spring-boot:run
```

### On a Windows environment in the root directory
```powershell
> mvnw.cmd spring-boot:run
```
