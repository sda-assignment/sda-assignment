User
- email (PK)
- username
- password
- isStaff
- wallet

Service
- name (PK)

Transaction
- id
- email
- timestamp
- amount
- type

ServiceProviderTransaction
- transactionId (PK, FK)
- serviceName (FK)
- providerName (FK)

Provider
- serviceName (PK) (Ex: Recharge)
- providerName: string (PK) (Ex: Etisalat)
- cashOnDelivery: boolean

FormElement
- name (PK) (what is sent to the provider server is going to be {name: info}, for example {"phone": "+201239813232"})
- serviceName (PK, FK)
- providerName (PK, FK)
- type: FormElementType
- info: String
- hasDeductionAmount: Boolean
(there must only be one FormElement per serviceName and providerName that his this field set to true.
Also when set to true, the info must be an integer that is going to be deducted from the wallet)

Discounts
- id (PK)
- type
- percentage

UsedDiscount
- email (PK)
- discountId (PK)

SpecificDiscount
- discountId (pk, fk)
- serviceName (FK)
- providerName (FK)

RefundRequest
- id (PK)
- amount
- transactionId
- status
- userEmail
